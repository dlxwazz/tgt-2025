WHAT IS THE GAMBLER TOKEN?

  • Gambler Network is an ethereum erc-20 based play-win crypto token project that aims to create virtual slot machines using decentralized applications.
  • THE GAMBLER aims to create a decentralized game of chance with transparent and simple rules .
  • Gambler uses decentralized applications to create a public virtual slot machine for its users. This virtual slot machine has 7 reels side by side, and each reel has 7 different fruit images.
  • Players start the game with the start button by placing bets of 1 tGt token for each game. After the reels stop, if the symbols match in a specified combination, the player wins the prize.

How to play?

  • The gambler will build his slot game on decentralized applications. Every player can play the decentralized virtual slot (dapp) game by obtaining a Virtual ethereum blockchain wallet.
  • The gameplay is simple and easy, similar to other slot games.There are 7 reels in the game. There are pineapple, pear, banana, apple, strawberry, watermelon and grape symbols on each reel. Players play each game with 1 tGt token.
  • Players win prizes when 3,4,5,6,7 of the same fruits line up after the reels stop. It does not matter which fruit or how they are listed.
  • But when 3 identical fruits are aligned in the E category, these 3 fruits must be the same as the fruit on the first reel.
  • For example, if there is a banana symbol on the first reel and 2 more bananas are aligned on the other reels, the player wins a prize. There is a banana symbol on the first reel, and the player cannot win a prize when there are 3 strawberry, 2 apple and 1 pear symbols on the other reels.

Prize distribution

Players instantly see the probability of winning and the total liquidity in each game. The tokens in the games played by all players are transferred to the common prize pool. The reward that the winning player will receive is calculated proportionally according to the total number of tokens in the liquidity pool and distributed automatically.

The gambler network does not promise a clear reward to the players in the game. Instead, players win the prize pro rata across 5 different categories. The aim here is to create a peer-to-peer game of chance by eliminating the need for a central management of the game. Prize rates will be ultimately decided taking into account sustainability in the game.

Scalability

The team is aware that one of the key risks for the project is scalability.But the team,It plans to minimize possible risks in this regard by taking an approach such as:

  • By getting rid of unnecessary data load
  • competitive and encouraging node system
  • simple and fluid game design

Applicability

Although many blockchain projects are very good ideas, these projects have serious problems in applicability.Serious problems arise in the integration of these projects, which require very large data recording ranges, into blockchain.

With the simple game structure and low data generation of slot games, it is easily integrated into decentralized applications. The virtual slot machine and gambler network, created using its unique reward distribution system and decentralized applications, will provide users with a safe, efficient and unique user experience.

Security

The decentralized nature of dapps ensures there is no single point of failure, making them more resilient to downtime and certain types of hacking attempts.

The transparent and immutable nature of blockchain technology ensures data integrity and reduces the likelihood of certain types of transaction fraud or manipulation.

Dapps give users more control over their data and digital assets. Since there is no central authority controlling the application, users can have the freedom to manage and transfer their data as they see fit.
